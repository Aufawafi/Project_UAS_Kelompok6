BAB IX
TREN & ARAH PERKEMBANGAN WEB

9.1 Evolusi HTML, CSS, dan JavaScript Terbaru

Teknologi web terus berkembang dengan fitur-fitur baru yang membuat pengembangan web semakin powerful dan efisien.

HTML5 Modern

  • Web Components & Shadow DOM
  • WebRTC untuk real-time communication
  • Web Workers untuk multi-threading
  • Service Workers untuk offline capability

CSS Modern

  • CSS Grid Level 2 & Subgrid
  • Container Queries
  • CSS Custom Properties (Variables)
  • CSS Houdini untuk custom rendering

JavaScript Modern (ES2024+)

  • Optional Chaining & Nullish Coalescing
  • Top-level await
  • Private fields untuk classes
  • Temporal API untuk date/time

9.2 Framework Modern dalam Pengembangan Web

Framework modern membantu developer membangun aplikasi web yang kompleks dengan lebih cepat dan terstruktur.

Frontend Frameworks

  • React: Library populer dari Meta untuk building UI dengan component-based architecture
  • Vue.js: Progressive framework yang mudah dipelajari dan sangat flexible
  • Angular: Full-featured framework dari Google untuk enterprise applications
  • Svelte: Compiler-based framework dengan performance tinggi

Meta-Frameworks

  • Next.js: React framework dengan SSR & Static Generation
  • Nuxt.js: Vue framework untuk server-side rendering
  • SvelteKit: Framework untuk Svelte applications

Backend for Frontend

  • Node.js: JavaScript runtime untuk backend
  • Express.js: Minimalist web framework
  • Nest.js: TypeScript framework dengan architecture yang solid

9.3 Progressive Web Apps (PWA) sebagai Tren Baru

PWA adalah aplikasi web yang memberikan pengalaman seperti native app dengan menggunakan teknologi web modern.

Karakteristik PWA

  • Progressive: Berfungsi di semua browser dengan progressive enhancement
  • Responsive: Fit di berbagai form factor (desktop, mobile, tablet)
  • Offline-First: Bisa berfungsi offline dengan Service Workers
  • App-like: Feel seperti native app dengan app shell model
  • Installable: Bisa diinstall ke home screen tanpa app store
  • Re-engageable: Push notifications untuk re-engagement

Teknologi PWA

  • Service Workers: Proxy antara web app dan network
  • Web App Manifest: JSON file yang define app metadata
  • HTTPS: Secure connection adalah requirement
  • Cache API: Untuk offline storage

Contoh PWA Terkenal

  • Twitter Lite
  • Pinterest
  • Starbucks
  • Spotify Web Player
