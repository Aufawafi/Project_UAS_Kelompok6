BAB V
DOM DAN MANIPULASI ELEMEN

Pada bab sebelumnya, kita telah mempelajari dasar-dasar sintaks JavaScript seperti variabel, tipe data, dan fungsi. Namun, kode JavaScript tersebut hanya berjalan di konsol dan belum benar-benar berinteraksi dengan tampilan halaman web. Di bab ini, kita akan masuk ke inti dari pengembangan Front-End, yaitu DOM (Document Object Model). DOM adalah jembatan yang menghubungkan JavaScript dengan HTML dan CSS, memungkinkan kita untuk membuat halaman web yang dinamis, interaktif, dan responsif terhadap tindakan pengguna.

5.1 Konsep Document Object Model (DOM)

Document Object Model atau disingkat DOM adalah representasi terstruktur dari dokumen HTML yang dibuat oleh browser saat halaman web dimuat. Sederhananya, DOM adalah "peta" atau model dari seluruh elemen yang ada di halaman web Anda. Browser membaca kode HTML baris demi baris dan mengubahnya menjadi serangkaian objek yang tersusun dalam struktur pohon (tree structure).

Struktur Pohon DOM (DOM Tree)

Dalam struktur pohon ini, setiap bagian dari dokumen disebut sebagai Node (simpul):

  • 1. Document Node: Akar utama dari pohon, mewakili seluruh halaman web.
  • 2. Element Node: Mewakili tag HTML (seperti <body>, <h1>, <div>, <p>).
  • 3. Text Node: Mewakili isi teks di dalam elemen (misalnya tulisan "Halo Dunia" di dalam <h1>).
  • 4. Attribute Node: Mewakili atribut pada tag HTML (seperti id, class, src, href).

Hubungan Antar Node

Hubungan antar node ini digambarkan seperti silsilah keluarga:

  • Parent (Induk): Elemen yang membungkus elemen lain. Contoh: <body> adalah parent dari <h1>.
  • Child (Anak): Elemen yang berada di dalam elemen lain.
  • Sibling (Saudara): Elemen-elemen yang berada di dalam parent yang sama.

Dengan memahami konsep ini, JavaScript dapat "berjalan-jalan" menelusuri pohon DOM untuk menemukan elemen tertentu, lalu mengubah, menghapus, atau menambahkannya secara dinamis tanpa perlu memuat ulang (reload) halaman.

5.2 Mengakses Elemen HTML dengan getElementById dan querySelector

Langkah pertama dalam memanipulasi DOM adalah "menangkap" atau menyeleksi elemen yang ingin kita ubah. JavaScript menyediakan beberapa metode untuk melakukan hal ini, mulai dari cara klasik hingga cara modern yang lebih fleksibel.

5.2.1 Menggunakan ID (getElementById)

Ini adalah metode paling cepat dan spesifik. Karena aturan HTML mengharuskan setiap ID bersifat unik (hanya boleh ada satu ID yang sama dalam satu halaman), metode ini akan selalu mengembalkan satu elemen tunggal.

HTML + JavaScript 
<!DOCTYPE html>
<html>
<head>
  <title>Mengakses dengan ID</title>
</head>
<body>
  <h1 id="judul">Selamat Datang</h1>
  <button onclick="ubahJudul()">Ubah Judul</button>

  <script>
    function ubahJudul() {
      // Tangkap elemen dengan ID "judul"
      let elemenJudul = document.getElementById('judul');
      
      // Ubah teksnya
      elemenJudul.textContent = 'Judul Telah Diubah!';
    }
  </script>
</body>
</html>

5.2.2 Menggunakan Selektor CSS (querySelector)

Ini adalah metode modern yang sangat powerful. querySelector memungkinkan kita memilih elemen menggunakan sintaks yang sama persis dengan CSS Selector (bisa berdasarkan ID, Class, Tag, atau kombinasi atribut). Metode ini akan mengembalikan elemen pertama yang ditemukan jika ada banyak elemen yang cocok.

HTML + JavaScript 
<!DOCTYPE html>
<html>
<head>
  <title>Mengakses dengan querySelector</title>
</head>
<body>
  <p class="intro">Ini adalah paragraf pertama</p>
  <p class="intro">Ini adalah paragraf kedua</p>
  <button onclick="ubahParagraf()">Ubah Paragraf</button>

  <script>
    function ubahParagraf() {
      // Pilih elemen pertama dengan class "intro"
      let p = document.querySelector('.intro');
      p.textContent = 'Paragraf telah diubah dengan querySelector!';
      
      // Atau gunakan ID
      // let el = document.querySelector('#judul');
      
      // Atau kombinasi
      // let el = document.querySelector('div.container p');
    }
  </script>
</body>
</html>

5.2.3 Menggunakan querySelectorAll (Banyak Elemen)

Jika kita ingin memilih semua elemen yang cocok dengan kriteria tertentu (misalnya memilih semua tombol di halaman), kita menggunakan querySelectorAll. Hasilnya bukan satu elemen, melainkan sebuah daftar (NodeList) yang mirip dengan Array.

HTML + JavaScript 
<!DOCTYPE html>
<html>
<head>
  <title>Mengakses Banyak Elemen</title>
</head>
<body>
  <button class="btn">Tombol 1</button>
  <button class="btn">Tombol 2</button>
  <button class="btn">Tombol 3</button>
  <button onclick="ubahSemuaTombol()">Ubah Semua</button>

  <script>
    function ubahSemuaTombol() {
      // Pilih SEMUA elemen dengan class "btn"
      let tombolList = document.querySelectorAll('.btn');
      
      // Loop untuk mengubah satu per satu
      tombolList.forEach(function(tombol, index) {
        tombol.textContent = 'Tombol ' + (index + 1) + ' diubah!';
        tombol.style.backgroundColor = 'green';
        tombol.style.color = 'white';
      });
    }
  </script>
</body>
</html>

5.3 Mengubah Konten dan Atribut Elemen

Setelah berhasil menangkap elemen, langkah selanjutnya adalah memanipulasinya. Kita bisa mengubah teks, menyisipkan HTML baru, mengganti gambar, atau bahkan mengubah gaya visual (CSS) secara langsung.

5.3.1 Manipulasi Konten (innerText vs innerHTML)

  • innerText atau textContent digunakan untuk mengubah teks polos di dalam elemen. Ini aman dari serangan XSS (Cross Site Scripting) karena tidak mengeksekusi tag HTML.
  • innerHTML digunakan jika kita ingin menyisipkan elemen HTML baru (seperti menambahkan <b>tebal</b> atau tag <span>).
HTML + JavaScript 
<!DOCTYPE html>
<html>
<body>
  <div id="kotak">Teks awal</div>
  <button onclick="gantiText()">Ganti Teks</button>
  <button onclick="sisipHTML()">Sisip HTML</button>

  <script>
    function gantiText() {
      let kotak = document.getElementById('kotak');
      // Mengubah teks polos (aman)
      kotak.textContent = 'Teks telah diganti!';
    }

    function sisipHTML() {
      let kotak = document.getElementById('kotak');
      // Menyisipkan HTML (bisa dangerous jika dari user input)
      kotak.innerHTML = '<b>Teks Tebal</b> dan <i>Miring</i>';
    }
  </script>
</body>
</html>

5.3.2 Manipulasi Atribut (setAttribute, src, href)

Kita sering perlu mengubah atribut elemen, misalnya mengganti gambar pada slider, atau mengubah tujuan link.

HTML + JavaScript 
<!DOCTYPE html>
<html>
<body>
  <img id="gambar" src="foto1.jpg" alt="Foto" />
  <a id="link" href="https://google.com">Link</a>
  <button onclick="ubahAtribut()">Ubah Atribut</button>

  <script>
    function ubahAtribut() {
      // Mengubah src gambar
      let img = document.getElementById('gambar');
      img.src = 'foto2.jpg';
      img.alt = 'Foto Baru';
      
      // Mengubah href link
      let link = document.getElementById('link');
      link.href = 'https://youtube.com';
      link.textContent = 'Link YouTube';
      
      // Menggunakan setAttribute (lebih umum)
      img.setAttribute('src', 'foto3.jpg');
      img.setAttribute('title', 'Gambar telah diubah');
    }
  </script>
</body>
</html>

5.3.3 Manipulasi Style dan Class (Penting!)

Mengubah tampilan elemen bisa dilakukan dengan dua cara:

  • style property, untuk menulis CSS inline (tidak disarankan untuk perubahan besar).
  • classList, untuk menambah atau menghapus class CSS yang sudah didefinisikan di file .css. Ini adalah cara terbaik (Best Practice) karena memisahkan logika (JS) dan gaya (CSS).
HTML + JavaScript + CSS 
<!DOCTYPE html>
<html>
<head>
  <style>
    .highlight {
      background-color: yellow;
      font-weight: bold;
      padding: 10px;
    }
  </style>
</head>
<body>
  <p id="teks">Ini adalah teks biasa</p>
  <button onclick="ubahStyle()">Ubah Style</button>
  <button onclick="tambahClass()">Tambah Class</button>

  <script>
    function ubahStyle() {
      let p = document.getElementById('teks');
      // Mengubah CSS inline (tidak disarankan untuk banyak properti)
      p.style.color = 'red';
      p.style.fontSize = '24px';
      p.style.backgroundColor = 'lightblue';
    }

    function tambahClass() {
      let p = document.getElementById('teks');
      // Menambah class CSS (Best Practice!)
      p.classList.add('highlight');
      
      // Menghapus class
      // p.classList.remove('highlight');
      
      // Toggle class (tambah jika tidak ada, hapus jika ada)
      // p.classList.toggle('highlight');
    }
  </script>
</body>
</html>

5.4 Event Listener dan Interaktivitas Halaman

Halaman web statis itu kurang menarik, kita ingin halaman bereaksi ketika pengguna melakukan sesuatu. Di sinilah Event berperan. Event adalah kejadian yang terjadi di browser, seperti pengguna mengklik mouse, menekan tombol keyboard, menggerakkan mouse (hover), atau mengirim formulir (submit).

Menggunakan addEventListener

Cara modern dan standar untuk menangani event adalah menggunakan metode addEventListener. Metode ini lebih unggul dibandingkan atribut HTML kuno (seperti onclick="...") karena memungkinkan kita memisahkan kode JavaScript dari HTML sepenuhnya.

Sintaks dasarnya:

elemen.addEventListener('jenis_event', fungsi_callback);

Jenis-Jenis Event Populer:

  • Mouse Events: click, dblclick, mouseover, mouseout
  • Keyboard Events: keydown, keyup, keypress
  • Form Events: submit, change, focus, blur
  • Window Events: load, resize, scroll
HTML + JavaScript 
<!DOCTYPE html>
<html>
<body>
  <button id="tombol">Klik Saya</button>
  <div id="hasil"></div>

  <script>
    // Tangkap elemen
    let tombol = document.getElementById('tombol');
    let hasil = document.getElementById('hasil');
    let counter = 0;

    // Tambahkan event listener untuk event 'click'
    tombol.addEventListener('click', function() {
      counter++;
      hasil.textContent = 'Tombol sudah diklik ' + counter + ' kali';
      
      if (counter >= 5) {
        tombol.style.backgroundColor = 'green';
        tombol.textContent = 'Sudah 5 kali!';
      }
    });

    // Event mouseover
    tombol.addEventListener('mouseover', function() {
      tombol.style.cursor = 'pointer';
      tombol.style.transform = 'scale(1.1)';
    });

    // Event mouseout
    tombol.addEventListener('mouseout', function() {
      tombol.style.transform = 'scale(1)';
    });
  </script>
</body>
</html>

5.5 Contoh Sederhana (Studi Kasus)

Sebagai penutup bab ini, mari kita gabungkan semua konsep di atas (Seleksi, Manipulasi, dan Event) untuk membuat dua fitur umum: Toggle Menu Navigasi (untuk tampilan responsif) dan Form Validasi Sederhana.

Studi Kasus: Toggle Menu (Navigasi Mobile)

Fitur ini umum digunakan pada desain web responsif. Saat di layar kecil, menu disembunyikan dan hanya muncul tombol garis tiga. Ketika tombol diklik, menu muncul.

Kode HTML:

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Toggle Menu - Studi Kasus</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <nav class="navbar">
    <div class="navbar-container">
      <div class="logo">MyWebsite</div>
      
      <!-- Hamburger Icon (akan muncul di mobile) -->
      <button class="hamburger" id="hamburger">
        <span></span>
        <span></span>
        <span></span>
      </button>
      
      <!-- Menu Navigasi -->
      <ul class="nav-menu" id="navMenu">
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#services">Services</a></li>
        <li><a href="#contact">Contact</a></li>
      </ul>
    </div>
  </nav>
  
  <main>
    <h1>Selamat Datang di Website Saya</h1>
    <p>Lihat menu navigasi bekerja di berbagai ukuran layar!</p>
  </main>
  
  <script src="script.js"></script>
</body>
</html>

Kode JavaScript:

JavaScript 
// Tangkap elemen hamburger dan nav menu
const hamburger = document.getElementById('hamburger');
const navMenu = document.getElementById('navMenu');

// Event listener untuk toggle menu
hamburger.addEventListener('click', function() {
  // Toggle class 'active' pada hamburger dan menu
  hamburger.classList.toggle('active');
  navMenu.classList.toggle('active');
});

// Optional: Tutup menu ketika link diklik (UX lebih baik)
const navLinks = document.querySelectorAll('.nav-menu li a');

navLinks.forEach(function(link) {
  link.addEventListener('click', function() {
    // Hapus class 'active' untuk menutup menu
    hamburger.classList.remove('active');
    navMenu.classList.remove('active');
  });
});

Penjelasan Studi Kasus:

HTML: Struktur navbar dengan hamburger button dan menu list.

CSS: Desain responsif dengan media query @768px, menu disembunyikan dengan max-height: 0 dan ditampilkan dengan max-height: 300px saat class active ditambahkan.

JavaScript: Menggunakan addEventListener dan classList.toggle() untuk menambah/menghapus class active pada hamburger dan menu.

Best Practice: Pisahkan HTML (struktur), CSS (tampilan), dan JavaScript (logika) untuk kode yang lebih bersih dan mudah di-maintain!
