DAFTAR PUSTAKA

Berikut adalah daftar referensi yang digunakan dalam penyusunan materi Tutorial Pemrograman Web 1. Seluruh sumber telah dipilih berdasarkan validitas, kredibilitas, dan relevansinya dengan materi pembelajaran pengembangan web fundamental.

Buku Teks

Duckett, J. (2014). HTML & CSS: Design and build websites. Wiley.

Buku comprehensive yang menjelaskan dasar-dasar HTML dan CSS dengan pendekatan visual yang mudah dipahami. Sangat cocok untuk pemula yang ingin memahami struktur dan styling website.

Duckett, J. (2015). JavaScript and jQuery: Interactive front-end web development. Wiley.

Panduan lengkap JavaScript dan jQuery dengan fokus pada pengembangan front-end interaktif. Buku ini membahas konsep fundamental JavaScript hingga implementasi praktis dalam proyek web.

Dokumentasi Resmi

Mozilla Developer Network. (2025, January 15). HTML: HyperText Markup Language. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/HTML

Dokumentasi resmi dan komprehensif tentang HTML dari Mozilla. Sumber terpercaya untuk referensi elemen HTML, atribut, dan best practices dalam pengembangan web semantik.

Mozilla Developer Network. (2025, February 10). JavaScript guide. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide

Panduan lengkap JavaScript yang mencakup syntax, konsep dasar, hingga fitur-fitur modern ECMAScript. Referensi utama untuk memahami JavaScript secara mendalam.

W3Schools. (2025). CSS tutorial. https://www.w3schools.com/css/

Tutorial interaktif CSS yang menyediakan contoh-contoh praktis dan editor online untuk latihan langsung. Sangat berguna untuk pembelajaran hands-on CSS properties dan selectors.

Framework & Library Modern

React. (2024). React documentation. https://react.dev

Dokumentasi resmi React library untuk membangun user interface berbasis komponen. Sumber referensi untuk memahami konsep component-based architecture dan Virtual DOM.

Vue.js. (2024). Vue.js guide. https://vuejs.org/guide

Panduan resmi Vue.js, framework progresif untuk membangun user interface. Dokumentasi yang sangat well-structured dengan contoh-contoh praktis untuk berbagai use case.

Svelte. (2024). Svelte tutorial. https://svelte.dev/tutorial

Tutorial interaktif Svelte yang memperkenalkan pendekatan compile-time framework. Memberikan pemahaman tentang paradigma baru dalam pengembangan web modern.

Next.js. (2024). Next.js documentation. https://nextjs.org/docs

Dokumentasi lengkap Next.js framework untuk React dengan fitur server-side rendering dan static site generation. Referensi untuk memahami optimasi performa web application.

Tutorial & Panduan Web Modern

Google Developers. (2024). Progressive web apps. https://developers.google.com/web/progressive-web-apps

Panduan resmi dari Google tentang Progressive Web Apps (PWA). Menjelaskan konsep, teknologi pendukung (service workers, web app manifest), dan implementasi PWA untuk menciptakan pengalaman web yang menyerupai aplikasi native.

Catatan Penggunaan Referensi:

Seluruh referensi di atas telah diakses dan diverifikasi keakuratannya pada periode penyusunan materi (2024-2025). Untuk referensi online, sangat disarankan untuk selalu mengecek pembaruan terbaru karena dokumentasi web framework dan library sering mengalami update seiring perkembangan teknologi.

Tips untuk mahasiswa: Manfaatkan dokumentasi resmi (MDN, W3Schools) sebagai referensi utama untuk memperdalam pemahaman konsep fundamental. Framework- framework modern dapat dipelajari setelah menguasai dasar HTML, CSS, dan JavaScript dengan solid.
