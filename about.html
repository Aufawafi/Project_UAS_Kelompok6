Tentang Project Ini

Deskripsi Project

Website tutorial ini dibuat sebagai bagian dari Project UAS Mata Kuliah Pengantar Pemrograman Web 1 di Universitas Pamulang. Project ini bertujuan untuk mendemonstrasikan pemahaman komprehensif tentang teknologi web fundamental.

Materi yang disajikan dalam website ini diambil dari makalah yang telah kami susun sebelumnya, yang mencakup pembahasan lengkap tentang HTML5, CSS3, dan JavaScript modern (ES6+).

Tujuan Pembelajaran

  • Memahami sejarah dan arsitektur World Wide Web
  • Menguasai HTML5 dan elemen semantik
  • Mampu membuat styling yang menarik dengan CSS3
  • Menggunakan Flexbox dan Grid untuk layout modern
  • Menguasai JavaScript untuk interaktivitas
  • Mengimplementasikan responsive web design
  • Menerapkan best practices dalam web development
